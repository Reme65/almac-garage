# Almac & Sons Vehicle Repairs Website

## Project Overview
This website was created as Project 1 for the Code Institute Diploma in Web Application Development.
The project demonstrates the use of HTML5 and CSS3 to create a responsive, multi-page static website.

The site represents a family-run local garage offering MOT testing, vehicle servicing and repairs.

---

## User Experience (UX)

### Target Audience
- Local vehicle owners
- Customers looking for reliable MOT and servicing
- Families seeking a trustworthy local garage

### User Goals
- View services offered
- Learn about the business
- Contact the garage easily
- Book an appointment

### Design Decisions
- Consistent navigation across all pages
- Page-specific header colours to improve orientation
- Simple, clear layout for accessibility
- Professional colour palette to build trust

---

## Features
- Three-page website (About, Services, Booking)
- Booking form with validation
- Clear contact details
- Responsive layout
- Semantic HTML structure
